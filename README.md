# Franca-IDL-syntax-higlight-st3
Franca IDL syntax higlight st3

## Installation

Ctrl+Shift+P 

```
pcar 
press Enter
put https://github.com/arthurfait/Franca-IDL-syntax-higlight-st3.git to the Repository URL
```

Ctrl+Shift+P
```
pci
search for franca highlighting
```

![sel](/install/selection.png?raw=true)

Install it.

**Enjoy.**
